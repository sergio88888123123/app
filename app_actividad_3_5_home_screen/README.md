# Actividad 3.5 - Home Screen

## Descripción

Este proyecto corresponde a la actividad **3.5 Home screen** de Flutter.

El objetivo de la actividad es diseñar la pantalla de inicio de una aplicación móvil utilizando widgets de layout. La pantalla principal muestra un mensaje de bienvenida, el nombre de la aplicación, una imagen de fondo y un ícono.

## Elementos solicitados

- Mensaje en pantalla: **Hello World** y bienvenida.
- Nombre de la aplicación: **Catálogo de Películas**.
- Imagen de fondo integrada con `Image.asset`.
- Ícono integrado con el widget `Icon`.
- Uso de widgets de layout como `Scaffold`, `Stack`, `Container`, `Column`, `Padding`, `SafeArea` y `Text`.

## Archivos principales

- `lib/main.dart`: contiene el diseño de la pantalla de inicio.
- `assets/images/fondo_home.png`: imagen utilizada como fondo.
- `pubspec.yaml`: configuración del asset utilizado por Flutter.

## Cómo ejecutar el proyecto

```bash
flutter pub get
flutter run
```

## Entrega

El proyecto se entrega mediante el enlace del repositorio en GitHub, como control de versiones solicitado en la actividad.
