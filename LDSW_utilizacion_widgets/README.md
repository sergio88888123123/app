# LDSW - Actividad 3.4 Utilización de Widgets

## Descripción

Este proyecto corresponde a la actividad **3.4 Utilización de widgets**.

El objetivo de la actividad es desarrollar una aplicación móvil básica en Flutter utilizando los widgets principales:

- Text
- Row
- Column
- Stack
- Container

La aplicación muestra una pantalla principal con una tarjeta visual donde se integran estos widgets de forma funcional y ordenada.

## Widgets utilizados

### Text
Se utiliza para mostrar títulos, descripciones y etiquetas dentro de la interfaz.

### Row
Se utiliza para organizar tarjetas pequeñas de widgets en forma horizontal.

### Column
Se utiliza para organizar los elementos principales de la pantalla de forma vertical.

### Stack
Se utiliza para sobreponer elementos visuales, como círculos e íconos.

### Container
Se utiliza para crear bloques visuales con color, tamaño, padding, bordes redondeados y sombras.

## Requisitos

Para ejecutar este proyecto es necesario tener instalado:

- Flutter SDK
- Dart SDK
- Visual Studio Code o Android Studio
- Un emulador Android o dispositivo físico

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

2. Entrar a la carpeta del proyecto:

```bash
cd LDSW_utilizacion_widgets
```

3. Instalar dependencias:

```bash
flutter pub get
```

4. Ejecutar la aplicación:

```bash
flutter run
```

## Evidencia de la actividad

El código entregado muestra la configuración correcta de los widgets solicitados y la aplicación funciona como una primera interfaz móvil desarrollada en Flutter.
